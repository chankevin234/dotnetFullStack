# dotnetFullStack
Learning how to use build an app from start to finish using .Net 7.0, React 18 (with Typescript), Node JS, and Mobx

Concepts:
- Entity Framework & SQLite
- CRUD Operations
- CQRS & Mediator Pattern

Tools:
- Axios
- SignalR
- Postman
- Semantic UI (CSS Framework)
- SQLite
- React Router
